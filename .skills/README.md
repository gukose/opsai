# Hotel OpAI Skills

These skills are the operational instructions for Hotel OpAI work.

Source of truth:

- `.handbook/` for product, architecture, workflow, task, AI, UniMock, testing, and standards
- `docs/ui-reference/` for UI implementation and visual matching

Skills:

- `architecture/` for system boundaries and clean architecture decisions
- `design-system/` for UI implementation against `docs/ui-reference/`
- `backend/` for Kotlin Spring Boot backend work
- `mobile/` for React Native and Expo work
- `workflow/` for conversation workflow orchestration
- `task-engine/` for task lifecycle and transitions
- `ai/` for AI conversation interpretation and fallback behavior
- `unimock/` for PMS simulator integration
- `testing/` for validation, test design, and release confidence
