# Design System

Hotel OpAI should look modern and premium while remaining operational and compact.

## Visual Direction

- Inspired by ChatGPT, Apple, Linear, and Notion
- restrained surfaces
- compact spacing
- clear hierarchy
- reusable visual primitives

## Rules

- Prefer reusable components over one-off screen elements.
- Keep borders, shadows, and radius consistent.
- Use one icon family across the app.
- Match reference images instead of reinterpreting them.
- Avoid decorative clutter that does not support workflow.

## Layout Rules

- Home screen stays compact.
- Only the assistant conversation scrolls.
- Composer remains pinned to the bottom.
- Task preview appears inline.
